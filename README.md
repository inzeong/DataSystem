# Energy-System 🏠

## About
가구 전력 통합 모니터링 시스템

## Version

Java : JDK1.8.0 <br>
Hadoop : hadoop-3.3.1 <br>
Kafka : kafka:2.12-2.0.1 <br>
Zookeeper : zookeeper:3.4.6 <br>

## Model 
ARIMA [Auto-regressive Integrated Moving Average] <br>
SARIMA+X [Seasonal ARIMA] <br>
LSTM [Long Short Term Memory]<br>
